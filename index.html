Minhas experiências

Minhas principais experiências foram dentro do ambiente acadêmico, lá participei de alguns projetos que considero importantes como:

Aplicativo gerenciador de tarefas

Aplicativo mobile desenvolvido na linguagem Kotlin com foco em organização e prioridade de tarefas. Neste projeto aprendemos sobre a metodologia ágil Scrum, realizamos sprint's semanais para compartilhar sobre o processo de desenvolvimento além de documentar com detalhes sobre o andamento do projeto. uma experiência bastante enriquecedora.

Gerador de contra-cheque

Sistema acadêmico criado para automatizar cálculos salariais e gerar contra-cheques. Aqui aproveitamos a disciplina de Banco de dados para desenvolver um sistema para calcular o contra-cheque de um funcionário, neste caso, por ser uma atividade acadêmica, calculamos todos os desconto utilizando SQL e, ao final do processo, era gerado um PDF utilizando Python

Participei de exposições de robótica em escolas na região de castanhal onde demonstramos o funcionamento de pequenos robôs utilizando de microcontroladores e placas de arduino junto de workshops para ensinar o básico do seu funcionamento.
